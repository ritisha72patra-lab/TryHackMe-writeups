# TryHackMe-writeups
TryHackMe is a great platform for practicing beginner level Capture the Flags (CTFs) which further help in develop real-world cybersecurity skills. This writeup consists of tools, technique and approaches to solve each challenge ranging from easy to hard.

## 🛠️ Tools & Techniques Used
- **Scanning:** Nmap, Gobuster
- **Password Cracking:** Hydra, John the Ripper
- **Connecting:** Netcat
- **Forensics:** Wireshark, Steghide
